# 💎 Premium React Login-Signup Application

A professional, high-performance, and visually stunning user authentication platform built with **React**, **Framer Motion**, and **Lucide React**. This project has been transformed from a basic template into a premium digital experience with modern UI/UX principles.

## ✨ Key Features

- **🛡️ Secure Simulated Authentication**
  - Modern Login & Signup flows with state persistence.
  - Real-time form validation and error handling.
  - Protected routing for authenticated users.

- **🎨 Premium UI/UX**
  - **Glassmorphism Design**: Sleek, transparent card layouts with backdrop blur.
  - **Dynamic Backgrounds**: Animated radial glow effects for a high-end feel.
  - **Micro-animations**: Smooth transitions and entrance effects using Framer Motion.
  - **Modern Typography**: Inter & Outfit Google Fonts for maximum readability.

- **📱 Fully Responsive**
  - Optimized for Mobile, Tablet, and Desktop.
  - Touch-friendly interactive elements.

- **⚡ Optimized Architecture**
  - Unified **Design System** with global CSS variables.
  - Optimized asset management (removed boilerplate/dead code).
  - Clean, modular component structure for better maintainability.

## 🛠️ Tech Stack

- **Core**: React 19
- **Navigation**: React Router 7
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Styling**: Modern Vanilla CSS with a custom Design System

## 🚀 Getting Started

1. **Clone & Enter**
   ```bash
   git clone https://github.com/Ukzzz/Login-Signup-Page.git
   cd Login-Signup-Page
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Launch the Experience**
   ```bash
   npm start
   ```
