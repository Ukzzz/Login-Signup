# Changelog

All notable changes to this project will be documented in this file.

## [1.1.0] - 2026-03-25

### Added
- **Design System**: Global CSS variables for consistent colors, typography, and spacing.
- **Glassmorphism UI**: Modern, transparent card layouts for Login and Signup forms.
- **Framer Motion**: Smooth entrance animations and page transitions.
- **Lucide React**: Professional icon set for enhanced visual communication.
- **Responsive Grid**: Improved form layout on the Signup page for better field organization.
- **Background Effects**: Dynamic radial gradients for a premium "dark mode" aesthetic.
- **Auth Simulation**: Basic authentication state management in `App.js`.

### Changed
- **Home Page**: Complete redesign with a landing page hero section and clear call-to-action buttons.
- **Login Page**: Transformed from a basic form to a premium glassmorphic experience.
- **Signup Page**: Redesigned for better UX and consistency with the Login page.
- **Project Structure**: Organized components and styles into a cleaner directory structure.
- **README.md**: Updated to reflect the new premium features and tech stack.

### Fixed
- **Security**: Removed hardcoded credentials from the main application logic.
- **UX**: Improved form validation feedback and added loading states.

### Removed
- Unused `logo.svg` asset.
- Boilerplate files: `reportWebVitals.js`, `setupTests.js`, `App.test.js`.
- Redundant CSS rules in `App.css` and individual component styles.
